Pro-tip: Reproducible research

Reproducible research is a hot topic, in part because a couple of high-profile papers that were disastrously non-reproducible (see "Deriving chemosensitivity from cell lines: Forensic bioinformatics and reproducible research in high-throughput biology").I also have pretty strong feelings about the subject ("Statisticians and computer scientists - if there is no code, there is no paper") When you write code for statistical analysis try to make sure that:

  1. It is neat and well-commented - liberal and specific comments are your friend

  2. That it can be run by someone other than you, to produce the same results that you report.
