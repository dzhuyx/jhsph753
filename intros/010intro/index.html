Summary feedback (part 1)

  1. Describe the x and y axis, what lines/points mean, and any trends that the plot is trying to communicate.

  2. Describe in concrete (read mathematical) terms the model that you are fitting.

  3. Report estimates, interpret the estimates, report measures of uncertainty on a scientific scale (read: confidence intervals/standard errors), and measures of statistical significance.

  4. If you use statistical methods that you haven't learned in this course that is great! But please write down the model you are applying rather than reference another text.

Summary feedback (part 2)

  1. Do not include raw output from R (or any other programming language) directly in the write-up of a report.

  2. Do not forget to answer specific questions. Even if one question is easier to answer/more interesting - be sure you provide a quantitative answer for every question asked.
